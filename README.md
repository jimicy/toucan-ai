# Toucan
AI-Powered Consulant for International Trade, Shipping, Logistics.

The app in this repo is deployed at [https://toucan-ai.onrender.com/](https://toucan-ai.onrender.com/).

<img width="2168" alt="Screenshot 2023-05-29 at 11 27 01 PM" src="https://github.com/jimicy/toucan-ai/assets/3952128/44941f75-ebd2-435f-afa8-2e8eb7794420">

Toucan AI provides support for 133 Languages that the consulant can reply to you in.
You can also speak to Toucan (speech-to-text) or play back (text-to-speech) in 133 languages as well.

<img width="1955" alt="Screenshot 2023-05-29 at 11 42 11 PM" src="https://github.com/jimicy/toucan-ai/assets/3952128/05aa323e-f126-4cc4-bb37-cc7d0ec41c2a">

Toucan also provides a Shipping Calculator for you to estimate your shipping rate from USPS, UPS, Fedex.
Currently there's a limitation in their API which only allows the sender country to be US, but the destination country can be any place.
